# Birthday Didi – Interactive Website

## What’s inside
- **index.html** – a single‑file, fully‑responsive site with all styling, animations, and scripts inlined.

## Open the site
1. Open **index.html** in any modern web browser (like Chrome or Edge) to view it.
2. No local server is required; you can just double-click the file!

## Personalize the letter
1. Open `index.html` in this editor.
2. Find the block starting around line 258:
```html
<!-- ✏️ REPLACE THE TEXT BELOW WITH YOUR PERSONAL BIRTHDAY LETTER ✏️ -->
<div class="letter-body">
  I'm not great with words...
</div>
```
3. Replace the text inside the `<div class="letter-body">` with your own message.
4. Save the file (Ctrl+S) and refresh the page in the browser to see your changes!
